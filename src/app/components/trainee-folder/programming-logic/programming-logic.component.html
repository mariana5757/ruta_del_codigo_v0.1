El iterador for permite ejecutar un bloque de instrucciones un determinado numero de veces, en este caso vamos a recorrer una lista de números para sumarlos.

En las listas empezamos en la posición 0, por eso tenemos que el for empieza en el valor 0 let i = 0 esta parte del for indica el valor en el que va a empezar el for, en este caso empieza en 0.

El for termina cuando la condición se cumple, en este caso el for termina cuando i sea menor que la longitud de la lista numbers.length.

El for incrementa la variable i en 1, en este caso i++.

Mientras vamos ejecutando el for, se va a ir acumulando la suma en la variable suma.

Realizando el mismo ejemplo pero esta vez con un while, vemos que la estructura es un poco diferente, pero cumple la misma función.

El while se ejecuta mientras la condición se cumpla, en este caso mientras i sea menor que la longitud de la lista numbers.length.

do-while es un poco diferente, en casos como for y while, se revisa ANTES de iterar si la condición de cumple, es decir que puede que nunca entre al ciclo. En el caso del do-while se ejecuta el bloque al menos una vez, y luego se revisa si la condición se cumple.

Es decir, en este caso realizará la primera suma sin importar si i es menor o mayor que la longitud de la lista numbers.length. Luego de realizar esa primera suma si empezamos a validar la condición.

Este método recorre cada elemento del arreglo numeros y ejecuta la función proporcionada por cada elemento.

Este ciclo recorre directamente las listas o arreglos, en este caso la variable number va a tomar el valor de cada elemento de la lista numbers.

Estos iteradores pueden ser usados para recorrer listas, arreglos o conjuntos. Por ejemplo map, filter, reduce, some, every, find, findIndex, findLast, findLastIndex, includes, sort, etc..

Estas funciones se usan para realizar operaciones en las listas o arreglos, como encontrar elementos, modificar ciertos elementos, eliminar, filtrar, etc..
