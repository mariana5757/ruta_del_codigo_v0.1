  • Bootstrap

    Bootstrap es un framework de código abierto para diseñar y desarrollar aplicaciones web responsivas y móviles. Fue creado por Twitter y es ampliamente utilizado por desarrolladores para agilizar la creación de interfaces web atractivas y funcionales.

    Maneja un sistema de grillas y una biblioteca de componentes preconstruidos, solo debemos hacer uso de sus clases para utilizar sus estilos.

    Principales caracteristicas de Bootstrap

    • Sistema de grillas, utiliza un sistema de 12 columnas que facilita la creación de diseños responsivos.
    • Componentes: Incluye una variedad de componentes predefinidos como botones, alertas, tarjetas, modales, formularios, y más.
    • Utilidades CSS: Provee clases para espaciado, alineación, colores, sombras, y otros estilos sin necesidad de escribir CSS personalizado.
    • JavaScript Integrado: Viene con plugins de JavaScript para agregar interactividad, como carruseles, menús desplegables y tooltips.
    • Personalización: Puedes personalizarlo usando variables Sass o sobrescribir sus estilos con CSS.

  • Material Design

    Material Design es una biblioteca de herramientas de código abierto para desarrollar interfaces de usuario basadas en HTML y CSS.
