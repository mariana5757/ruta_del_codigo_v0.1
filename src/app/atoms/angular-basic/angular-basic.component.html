Instalacción

Para instalar Angular necesitamos tener instalado también Node.js y npm(gestor de paquetes de Node). Para instalar node nos dirigimos a su sitio web oficial, allí buscaremos la última versión o la que el proyecto requiera. Node.js

Para Angular v18 o v19, tenemos como prerequisito node.js v^18.19.1 o la más nueva. Si necesitas otra versión puedes ver los prerequisitos en la documentación de Angular.

Cuando ya tengamos instalado Node.js y npm, abrimos el terminal y podemos instalar Angular CLI con el comando npm install -g @angular/cli

Crear un proyecto

Para crear un proyecto de Angular usamos el comando ng new nombre-del-proyecto
