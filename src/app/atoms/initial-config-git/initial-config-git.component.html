Para empezar a trabajar con GIT primero debemos registrarnos en GitHub

Luego de haber completado el registro debemos descargar git en nuestro equipo.

Cuando ya tengamos instalado git, lo debemos configurar con el correo que tenemos registrado en GitHub, vamos a buscar GitBash en nuestro equipo y hacemos uso de los siguientes comandos:

git config --global user.name "nombre de usuario"

git config --global user.email "correo registrado"

Estos comandos nos permiten configurar el correo y el nombre de usuario que usaremos en el proyecto, podemos verificar que todo haya quedado bien usando el siguiente comando:

git config --list
